# Python Assignment — Part 1: Python Basics & Control Flow

A command-line Student Grade Tracker built using core Python concepts.

## Tasks Covered

- **Task 1** — Data Parsing & Profile Cleaning
- **Task 2** — Marks Analysis using Loops & Conditionals
- **Task 3** — Class Performance Summary
- **Task 4** — String Manipulation Utility

## File

- `part1_grade_tracker.py` — contains all 4 tasks with comments and visible outputs

## How to Run
```bash
python part1_grade_tracker.py
```

## Topics Used

- String methods (strip, title, split, replace, count)
- Type conversion (int, list)
- for loops and while loops
- Conditionals (if/elif/else)
- f-strings and formatted output
- Input validation and error handling
